The Lost Temple Expedition - Unity Game

## Description
**The Lost Temple Expedition** is a first-person adventure game made in Unity. The player explores a mysterious jungle, collects 3 hidden artefacts, and enters an ancient temple filled with deadly traps. The goal is to survive and reach the end to achieve victory!

##  Game Structure

- **MainMenu Scene**: Start screen with options to Play, Quit, and adjust settings.
- **Jungle Scene**:
  - Explore and collect 3 artefacts.
  - After collecting all, a temple door opens.
  - Enter the temple to progress.
- **Temple Scene**:
  - Navigate through traps.
  - If the player dies → Game Over screen.
  - If the player survives → Victory screen.

## Features

- First Person Controller
- Artefact collection system
- Scene transition system
- Trap-triggered Game Over system
- Pause menu (ESC key)
- Audio control (Volume slider)
- UI Menus: Main Menu, Pause, Game Over, Victory
- Shaders: Basic and Water Shader (Shader Graph)
- URP compatible


## How to Play

1. Open the project in Unity.
2. Set the MainMenu scene as the starting scene.
3. Press **Play** 
4. Explore, collect, and survive!
